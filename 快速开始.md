# 快速开始指南

## 下载项目

从 Replit 下载所有文件到本地计算机。

## 在 Linux 环境中构建

### 方法一：本地 Ubuntu/Linux

```bash
# 1. 安装依赖（仅第一次）
sudo apt-get update
sudo apt-get install -y openjdk-17-jdk python3-pip git zip unzip
pip3 install --user buildozer cython==0.29.33

# 2. 进入项目目录
cd /path/to/parking-nav-app

# 3. 构建 APK
chmod +x build.sh
./build.sh

# 4. 等待构建完成（首次约30-60分钟）
# APK 文件将生成在 bin/ 目录
```

### 方法二：使用 Docker

```bash
# 1. 安装 Docker
# 参考: https://docs.docker.com/engine/install/

# 2. 进入项目目录
cd /path/to/parking-nav-app

# 3. 使用 Docker 构建
docker run --rm -v $(pwd):/app -w /app ubuntu:22.04 bash -c "\
  apt-get update && \
  apt-get install -y python3-pip openjdk-17-jdk git zip unzip && \
  pip3 install buildozer cython==0.29.33 && \
  buildozer android debug"

# 4. APK 文件将生成在 bin/ 目录
```

### 方法三：在线构建（GitHub Actions）

1. 将项目上传到 GitHub
2. 创建 `.github/workflows/build.yml`（参见 SETUP_GUIDE.md）
3. 推送代码触发自动构建
4. 下载构建好的 APK

## 安装到手机

### USB 安装

```bash
# 连接手机并启用 USB 调试
adb install bin/parkingnav-*.apk
```

### 手动安装

1. 将 APK 文件传输到手机（通过 USB、微信、邮件等）
2. 在手机上打开"设置" → "安全"
3. 启用"允许安装未知来源应用"
4. 使用文件管理器找到 APK 文件
5. 点击安装

## 使用应用

1. 打开"车位导航"应用
2. 先在系统蓝牙设置中配对目标设备
3. 回到应用，点击"扫描设备"
4. 选择设备后点击"连接设备"
5. 输入车位号（如 A11）
6. 点击"发送导航"

## 故障排查

### 构建失败
- 检查磁盘空间（至少 20GB）
- 删除 `.buildozer` 目录后重试
- 查看详细日志：`buildozer -v android debug`

### 无法连接蓝牙
- 确保设备已在系统中配对
- 检查蓝牙权限是否授予
- 重启应用和蓝牙

### 应用闪退
- 查看日志：`adb logcat | grep python`
- 检查 Android 版本（需要 5.0+）
